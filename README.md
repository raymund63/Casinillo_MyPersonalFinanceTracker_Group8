# Casinillo - My Personal Finance Tracker

![Python](https://img.shields.io/badge/Python-3.8+-blue?logo=python&logoColor=white)

A personal finance tracking application built with Python.

## Features

- Track financial records (expenses and income)
- Manage financial plans with due dates
- Data persistence with JSON storage
- Interactive command-line interface

## Requirements

- Python 3.8+

## Usage

Run the application:
```bash
python casinillo_mpft_8.py
```

## Project Structure

- `casinillo_mpft_8.py` - Main application file
